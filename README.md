## 🎬 Overview

**Kitchen Chaos** is a fast-paced, time-based cooking simulation game built in Unity.

You play as a chef inside a chaotic kitchen where speed, accuracy, and multitasking are essential.  
Your goal is to prepare and deliver orders before time runs out while managing multiple kitchen stations.

This project focuses on building a **complete gameplay loop**, interaction systems, and polished game feel.

---

## 🎯 Core Gameplay Loop


Explore Kitchen → Pick Ingredients → Process Food → Assemble Dish → Deliver Order → Repeat
