## 🎬 Overview

**Kitchen Chaos - Mobile Version** is a Unity-based cooking simulation game adapted for mobile devices.

This version focuses on transforming the original PC gameplay experience into a touch-friendly mobile UI, while keeping the same fast-paced core gameplay intact.

The goal is to maintain the original gameplay feel while **optimizing controls** and interface for **mobile screens**.

---

## 🎯 Core Gameplay Loop


`Explore Kitchen → Pick Ingredients → Process Food → Assemble Dish → Deliver Order → Repeat`

---

## ⚔️ Core Gameplay Systems (Unchanged)

- Smooth player movement system
- Ingredient pickup, carry, and drop mechanics
- Cooking & processing system
- Order generation & delivery system
- Timer-based gameplay loop
- **📱 Mobile-Specific Features**
- **🖐️ Touch Controls**
- On-screen buttons for interaction
- Mobile-friendly joystick/stick input
- Reworked input system for touch devices

### 📐 UI Adaptation
- Responsive UI scaling for different screen sizes
- Repositioned buttons for better accessibility
- Optimized layout for portrait/landscape support

### 🎧 Polish & Feel (Inherited)
- 🎵 Background music system
- 🔊 Sound effects for interactions
- 🎨 Visual feedback for actions

### 💥 Core game feel & responsiveness preserved
**🧪 Tech Stack**
*🎮 Unity Engine*
*💻 C#*
*📱 Unity Input System (Mobile Support)*
*🧠 UI/UX Adaptation Principles*

## 🚀 How to Run
- Install the [APK File](https://github.com/EngOmarAbd-Elaziz/UnityCourse/releases/tag/v1.0.1/Kitchen_Chaos_Mobile.apk) on Android device
- Launch the game
- Use on-screen controls to play 🍳🔥
